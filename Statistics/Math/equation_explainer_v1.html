Math / v1 Prototype

Equation Explainer

Text-first equation inspection built around LaTeX input, a local semantic parser, and MathJax rendering. This version focuses on render, color-by-structure, and explain-selected-subexpression. Audio is intentionally omitted.

Input

LaTeX In

Examples

Load a seeded equation, then edit it freely.

v1 Scope

Parser coverage is intentionally narrow and explicit.

  • Supported: fractions, sums, products, indices, brackets, roots, function calls, common operators.
  • Included: structure tree, semantic role coloring, selection-driven explanations.
  • Excluded: OCR, screenshot ingestion, external LLM calls, audio narration.
